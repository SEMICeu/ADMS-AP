ADMS Controlled Vocabularies

The ADMS Controlled Vocabularies contain a number of vocabularies with terms that can be used as objects in RDF properties, in particular with properties defined in the Asset Description Metadata Schema (ADMS).
These vocabularies were developed and are maintained under the SEMIC activity in the context of the European Commission's ISA and ISAÂ² Programmes. All distributions of the ADMS Controlled Vocabularies are available under the ISA Open Metadata Licence v1.1.
The following vocabularies are descibed below:

  • Asset Type
  • Change Type
  • Interoperability Level
  • Licence Type version 1.0
  • Licence Type version 2.0
  • Publisher Type
  • Representation Technique
  • Status
For further information and feedback, please contact contact@semic.eu.
Date of last modification: 30 March 2018

Available formats: HTML+RDFa, Turtle, RDF+XML
